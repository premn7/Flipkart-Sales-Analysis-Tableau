# Flipkart-Sales-Analysis-Tableau
The Flipkart Sales Analytics Dashboard analyzes e-commerce sales data using Tableau. It highlights key metrics like revenue, profit, and units sold. The dashboard includes sales trends, payment methods, and geographic insights across India, helping businesses understand customer behavior and support data-driven decisions.
🛒 Flipkart Sales Analytics Dashboard

This project is an interactive sales analytics dashboard designed to analyze business performance similar to the e-commerce platform Flipkart. The dashboard was developed using Tableau to transform raw sales data into clear visual insights for decision-making.

🎯 Project Objective

The main goal of this dashboard is to:

📊 Analyze overall sales performance
💰 Track revenue and profit trends
🛍️ Understand customer purchasing behavior
🌍 Identify regional sales distribution across India
🎉 Study the impact of major shopping events and festivals

📈 Key Performance Indicators (KPIs)

The dashboard highlights important business metrics:

💰 Total Revenue: ₹946.95M
📊 Total Profit: ₹147.10M
📦 Units Sold: 30,173

These KPIs help stakeholders quickly understand the overall business performance.

📊 Sales Trend Analysis

The dashboard includes a daily sales bar chart showing revenue patterns across the month.

✨ Purpose:

Identify high-sales periods

Detect sales fluctuations

Support better inventory planning

💳 Payment Method Analysis

The dashboard analyzes customer payment preferences:

💵 Cash on Delivery (COD)
💳 Credit Card
💳 Debit Card
🏦 EMI
📱 UPI

This helps understand how customers prefer to pay for products.

🌍 Geographic Sales Insights

A map visualization of India displays regional sales distribution.

📍 Benefits:

Identify top performing states

Understand regional market demand

Support location-based marketing strategies

🎉 Event & Festival Sales Analysis

The dashboard also evaluates sales during major events:

🎆 Republic Day
🎊 New Year
🇮🇳 Independence Day
🪔 Diwali
🛍️ Big Billion Days

These insights show how promotional events impact sales growth.

⚡ Interactive Dashboard Features

The dashboard allows users to interact with the data through:

🔘 Payment method filters
🎉 Event-based filters
📊 Dynamic charts and graphs

This makes it easy for users to explore insights quickly.

💡 Key Business Insights

From the analysis we can observe:

📈 Sales increase during festival seasons and promotional events
💳 Digital payment methods are widely used
🌆 Major urban states contribute significant revenue

🚀 Conclusion

This dashboard demonstrates how data visualization can transform raw data into actionable insights.

It helps businesses:

📊 Monitor performance
🎯 Identify opportunities
📈 Improve strategic decision-making

Overall, this project showcases data analysis, visualization, and business intelligence skills.
